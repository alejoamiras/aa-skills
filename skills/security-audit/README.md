# security-audit

Security audit for ecosystem projects — bridges, wallets, smart contracts — with special attention to **Solidity, Noir, and Aztec.nr** (reorgs, replay, front-running, oracle manipulation, reentrancy, censorship). Spawns parallel Claude + Codex agents for redundant coverage, then triages into a severity-rated report.

Complements [`harden`](../harden/): harden covers web/backend codebases; this skill owns the smart-contract domain. Protocol in [`SKILL.md`](SKILL.md).
