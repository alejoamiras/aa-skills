# wallet-sdk

Integration patterns for `@aztec/wallet-sdk`: wallet discovery, connection (ECDH + emoji verification), capability manifests, transactions and view calls — for dApp frontends and wallet extensions.

Includes [`migration-from-azguard.md`](migration-from-azguard.md), a side-by-side mapping from `@azguardwallet/client` to `@aztec/wallet-sdk` covering every integration pattern. API reference in [`wallet-sdk.md`](wallet-sdk.md), usage guidance in [`SKILL.md`](SKILL.md).
